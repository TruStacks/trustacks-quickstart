# dotnet-hello

Minimal ASP.NET Core service for the TruStacks quickstart workshop. The Code Reviewer agent identifies this as a C# / ASP.NET Core service; the DevOps Engineer emits CI + Helm + ArgoCD against it when you run `/plan`.

Run locally:

```sh
dotnet run
curl http://localhost:5000/
```
