# go-hello

Minimal Go service for the TruStacks quickstart workshop. The Code Reviewer agent identifies this as a Go service; the DevOps Engineer emits CI + Helm + ArgoCD against it when you run `/plan`.

Run locally:

```sh
go run .
curl http://localhost:8080/
```
