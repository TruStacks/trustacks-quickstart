# spring-boot-hello

Minimal Spring Boot service for the TruStacks quickstart workshop. The Code Reviewer agent identifies this as a Java / Spring Boot service; the DevOps Engineer emits CI + Helm + ArgoCD against it when you run `/plan`.

Run locally:

```sh
./mvnw spring-boot:run
curl http://localhost:8080/
```
